# Employee Management System

A **Java Console-Based Employee Management System** that allows users to manage employee records through a simple menu-driven interface. This project demonstrates the implementation of Object-Oriented Programming (OOP) concepts and basic CRUD operations using Java.

## 📖 Overview

The Employee Management System is designed to help users efficiently manage employee information. Users can add new employees, view all employee records, search employees by their ID, and update employee department details.

This project is suitable for beginners learning Java and OOP concepts.

---

## ✨ Features

- ➕ Add a new employee
- 📋 View all employees
- 🔍 Search employee by ID
- ✏️ Update employee department
- 🚪 Exit the application

## ▶️ How to Run

### 1. Clone the repository

```bash
git clone https://github.com/YOUR_GITHUB_USERNAME/EmployeeManagementSystem.git
```

### 2. Go to the project folder

```bash
cd EmployeeManagementSystem
```

### 3. Compile the Java files

```bash
javac *.java
```

### 4. Run the application

```bash
java Main
```
